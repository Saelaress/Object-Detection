Консольное приложение, которое позволяет искать на видео определенные объекты, вести трекинг и сохранять видео с ними в отдельный файл.
Используется YOLO для поиска объектов и openCV для манипуляций с видео.

Вход:

путь к файлу с исходным видео;
имя класса объекта (один из 80 классов, поддерживаемых YOLO).

Выход:

файл видео собранного из кадров, на которых вырезаны найденные объекты.
